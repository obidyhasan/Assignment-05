Important FAQ

What is the Document Object Model (DOM)?

Document Object Model (DOM) is a programming interface for HTML documents. When the webpage is loaded in the browser, the HTML document is converted into a DOM tree.

How do you select an element from the DOM?

There are many ways to select an element from the Document Object Model (DOM):

  • By Id - getElementById()
  • By class name: - getElementsByClassName()
  • By tag name: - getElementsByTagName()
  • By query selector: - querySelector()
  • By query selector all: - querySelectorAll()
    • What is event delegation in the context of the DOM, and why is it useful?

    Event Delegation is allows to handle events for multiple child elements efficiently without having to add individual event listeners to each one.


    Why is it useful?

    Efficiency: Event Delegation allows you to attach a single listener to the parent container. This results in less code and better performance.

    Dynamism: Event delegation is beneficial when dealing with dynamic content. As new elements are added or removed, the event listener remains attached to the parent, reducing the need for constant reattachment.

    How do you manipulate an element's attributes and styles using the DOM?

    Manipulating Attributes

  • getAttribute() - It's retrieves the value of a specified attribute.
  • setAttribute() - It's set the value of a specified attribute.
  • removeAttribute() - It's removes a specified attribute from the element.

    • Manipulating Style

  • Style property : It is provides direct access to the element's inline styles
